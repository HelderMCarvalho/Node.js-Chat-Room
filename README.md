# Node.js-Chat-Room
## Chat em tempo real com Node.js

###### Funcionalidades
- Identificação do utilizador que enviou a mensagem;
- Possibilidade de mensagens privadas entre utilizadores;
- Não divulgar a mensagem para o próprio emissor (acrescentar diretamente no cliente);
- Fazer registo e login com um utilizador;
- Dar a indicação de “o utilizador está a escrever”;
- Mostrar quais os utilizadores que estão atualmente ligados.

###### Como utilizar
É necessário criar um utilizador, fazer login com ele, e começar a enviar mensagens.
